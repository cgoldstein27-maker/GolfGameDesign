Study Smart

A browser-based study app: turn notes into flashcards, spaced repetition, quizzes, and progress tracking — with optional AI help.

Use ← → arrow keys or the buttons below to navigate

Overview

What does Study Smart do?

One place to build a personal study library and practice from it every day.

  • Library — Paste notes, upload .txt or PDF, or load AP-style starter topics.
  • Auto study pack — Each set gets a summary, chunked sections, and flashcards.
  • Study (SRS) — Spaced repetition (SM-2 style): cards come back when you’re due.
  • Quiz — Multiple-choice from your cards or AI-paraphrased questions from full notes.
  • Weak topics — See which sections you miss most; reset stats when you want a fresh start.
  • Ask notes — Chat grounded in your saved sets (optional OpenAI key).
  • Streak — Daily flame + week strip when you study (SRS or quiz).

Audience

Who is it for?

High school & AP students

  • Organize class notes by unit or period
  • Quick starter outlines for many AP courses
  • Review before tests without rebuilding Quizlet from scratch

Self-directed learners

  • Paste textbook excerpts or lecture notes
  • Track weak sections over time
  • Optional AI for harder quizzes and Q&A

Anyone who wants habit + memory

  • Short daily SRS sessions beat cramming
  • Streak encourages showing up each day
  • Everything stays on your device (localStorage)

Not a replacement for…

  • Official AP / College Board materials
  • A full cloud account across phones (device-local sign-in)
  • Proctored exam prep alone — it’s a practice companion

Benefits

How does it help students?

  • Saves time — Notes → flashcards + summary in one click; no manual card typing for every line.
  • Smarter review — SRS schedules “Again / Hard / Good / Easy” so hard cards return sooner.
  • Targets weak spots — Quiz misses and SRS failures feed the Weak topics tab.
  • Two study modes — Recall (SRS) and recognition (quiz) reinforce the same material differently.
  • Low friction — Runs in the browser; works on GitHub Pages or locally with a simple file server.
  • Privacy-friendly default — Core features work without an API key; AI is optional under the API tab.

Under the hood

What’s going on in the code?

Vanilla HTML/CSS/JS — no React. Modules in development; one bundled script for GitHub Pages and file://.

index.html → study-smart-bundle.js → localStorage

Core modules

  • storage.js — load/save state per user
  • study-engine.js — chunks, summaries, cards, quiz items
  • srs.js — SM-2 scheduling
  • weak-topics.js — miss/hit tallies
  • streak.js — daily streak + week ring
  • chat.js — Ask notes + AI quizzes
  • app.js — UI wiring (tabs, forms, lists)

Data flow (one save)

Paste notes → buildStudyMaterial()
  → chunks + flashcards + summary
  → state.docs[] + state.srs{}
  → persist() → localStorage

Rate card → scheduleReview()
  → update nextReview
  → recordWeak() if Again
  → applyStreakActivity()

Study engine

From text to flashcards

  • Notes split into chunks (paragraphs) — first line becomes the topic label.
  • Extractive summary — scores sentences by term overlap with the full doc.
  • Flashcards — question/answer pairs per chunk; distractors for multiple-choice quiz mode.
  • AP starters — ap-starter-topics.json loads course/topic outlines without upload.

Session & accounts

Email + password are hashed and stored in localStorage on this browser only. Each user gets their own state key (study-smart-v1::u-…). Sign-in is not a cloud backend — it separates profiles on one machine.

Habits & memory

SRS, streaks & insights

Spaced repetition

  • Each card stores ease factor, interval, and nextReview.
  • Study tab shows only due cards across all sets.
  • Quality 0–5 updates the schedule (Again resets progress).

Streak + week strip

  • One SRS rating or quiz answer counts for the day.
  • Consecutive calendar days build the streak count.
  • Su–Sa row shows ✓ done, missed days, and ☆ today.
  • UI follows system light/dark mode.

Optional AI

API tab (when you want more)

  • Ask notes — Retrieves relevant chunks locally; can call OpenAI for richer answers.
  • Conceptual quiz — Paraphrases your notes into new MC questions (needs key + enough text).
  • Advanced quiz — Notes + optional Wikipedia context via web-research.js.
  • Key stored only in localStorage on your device — never required for basic study.
